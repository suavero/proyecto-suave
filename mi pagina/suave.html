diseño de pagina web

elemento frontend

El frontend se refiere a todo lo que los usuarios ven y con lo que interactúan en la página web. Para hacer tu página interactiva, puedes usar herramientas y tecnologías como:

  • HTML: La estructura básica de la página.
  • CSS: Para el diseño y estilo de la página.
  • JavaScript (JS): Para la interactividad, como botones que muestran o esconden contenido, validación de formularios, animaciones, etc.

bases de datos

DBMS

Son las más comunes. Los datos se organizan en tablas con filas y columnas. Se utilizan lenguajes de consulta estructurados como SQL (Structured Query Language). Ejemplos incluyen:

  • mysql
  • postgresql
  • oracle
  • Microsoft SQL Server
